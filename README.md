# Music Genre Prediction using Decision Tree

## Overview
This project uses Machine Learning to predict a user's preferred music genre based on age and gender.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Graphviz
- Jupyter Notebook

## Features
- Data preprocessing
- Decision Tree Classification
- Model evaluation
- Decision Tree visualization

## Dataset
The dataset contains:
- Age
- Gender
- Music Genre

## How to Run
1. Open the notebook.
2. Install required libraries:
   pip install pandas scikit-learn graphviz
3. Run all cells.
